# IDL Problem Codes

Here you will find quick links/access to IDL's problem codes and their accompanying documentation.

If you are looking to turn off problems (i.e. don't report them), then you can use the IDL setting "IDL -> Problem reporting -> Ignore Problems" through the settings UI. This UI has auto-complete (with a description) of each problem.

> Not every problem code is documented yet, but if you think an important one is missing, let us know and we are happy to add it in.
